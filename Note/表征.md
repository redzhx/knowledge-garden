# 表征

表征（representation）是认知心理学的核心概念，当有机体对外界信息进行加工（输入、编码、转换、存储和提取等）时，这些信息是以表征的形式在头脑中出现的。  
认知表征是认知科学的核心问题。古希腊哲学家和现代认知哲学家、认知科学家都对这个问题进行过探讨，提出过理论，这些理论是基于某些隐喻提出假设，建立模型然后完成的，体现了认知科学研究的方法论意义。

如[[洞穴隐喻]]就是认知表征。
