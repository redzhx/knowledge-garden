
# 使用说明

_该模版为开智学堂信息分析课程信息分析报告通用 v1.0 版。_

_用于非专业场景下的个人报告，如读书笔记、研究作业等。_

# 信息报告标题

_标题应告知读者你的报告内容，从读者利益相关的角度拟写。_

# 0. 目录

_非必需。当信息报告篇幅较长时需要。_

_目录编写的目的是方便读者快速了解报告的结构，以及定位到特定的信息。_
